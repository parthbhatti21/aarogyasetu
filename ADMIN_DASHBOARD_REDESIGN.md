## 📊 **Metrics Calculation**

### Efficiency Score Calculation
```
Efficiency = (Patients Handled / Max Capacity) × 100
            + (Completion Rate / 100) × 30
            - (Avg Time vs Target) × 5
```

### Trend Calculation
```
Trend = Current Day Performance vs Previous Day
        ↑ = +3% or more (increasing)
        ↓ = -2% or more (decreasing)
        → = Between -2% and +3% (stable)
```

### Comparison Metrics
```
Improvement % = (Traditional - Current) / Traditional × 100
```
